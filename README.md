## Trabajos

### 1. Red_Neuronal_Iris_Flower
Red neuronal feedforward implementada desde cero para clasificar flores del dataset Iris en 3 especies.

**Carpeta:** [`Red_Neuronal_Iris_Flower/`](./Red_Neuronal_Iris_Flower)

**Características:**
- Implementación sin frameworks de alto nivel (NumPy, pandas, scikit-learn para preprocesado)
- Backpropagation manual
- Tests unitarios
- Documentación completa
